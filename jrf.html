/{court_code}/{story_identity}

Retrieve a specific story

/{court_code}/{story_identity} get

get

get /{court_code}/{story_identity}

  • Request
  • Response

URI Parameters

  • court_code: required (string - pattern: \w{3})

    法院代號

    Example:

    KSH
  • story_identity: required (string - pattern: .*-\d{2,3}-.+-\d+)

    案件識別碼 {story_type}-{story_year}-{story_word}-{story_number}

    Example:

    刑事-106-抗-99

Body

Media type: application/json

Type: object

Properties
  • story: required (object)
    • lawyer_names: (array of )

      相關律師姓名

    • identity: required (object)
      • type: required (string)

        案件類別

      • year: required (number)

        案件年份 民國年

      • word: required (string)

        案件字別

      • number: required (number)

        案件案號

    • court: required (object)
      • name: required (string)

        法院全名

        Example:

        臺灣雲林地方法院
      • code: required (string)

        法院代碼

        Example:

        ULD
    • prosecutor_names: (array of )

      相關檢察官姓名

    • adjudged_on: (date-only)

      判決日期

    • judges_names: (array of )

      相關法官姓名

    • reason: required (string)

      案由

    • verdict: required (object)
      • detail_url: required (string)

        判決書show 頁面連結

        Example:

        https://api.jrf.org.tw/KSH/%E5%88%91%E4%BA%8B-106-%E8%81%B2-397/verdict

      • detail_url: required (string)

        庭期表index 頁面連結

        Example:

        https://api.jrf.org.tw/KSH/%E5%88%91%E4%BA%8B-106-%E8%81%B2-397/schedules

      相關當事人姓名

    • pronounced_on: (date-only)

      宣判日期

Get a list of stories based on given condition

get /search/stories

Get a list of stories based on given condition

  • Request
  • Response

Query Parameters

  • page: (integer)

    Specify the page that you want to retrieve

    Example:

    1
  • q[story_type]: (string)

    案件類別

    Example:

    刑事
  • q[year]: (number)

    案件年份 民國年

    Example:

    105
  • q[word]: (string)

    案件字別

    Example:

    抗
  • q[number]: (number)

    案件案號

    Example:

    537
  • q[lawyer_names_cont]: (string)

    相關律師姓名包含

    Example:

    吳
  • q[judges_names_cont]: (string)

    相關法官姓名包含

    Example:

    張惠閔
  • q[adjudged_on_gteq]: (string)

    判決日期晚於

    Example:

    2009-12-01
  • q[adjudged_on_lteq]: (string)

    判決日期早於

    Example:

    2009-12-31
